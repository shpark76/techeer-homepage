Education Model

단계별 목표가 명확해야 성장이 보입니다

각 단계는 참가자가 무엇을 배우는지, 어떤 결과물을 만들어야 하는지, 다음 단계로 이동하기 위해 무엇을 증명해야 하는지 기준을 둡니다. 그래서 테커의 성장경로는 단순 수료 과정이 아니라 개발자로 일할 준비를 만드는 모델입니다.

Level 1

입문 진단과 몰입 준비

개발 경험과 관계없이 현재 수준을 확인하고, 5주 동안 몰입할 학습 루틴과 협업 태도를 정렬합니다.

목표
기초 역량과 성장 의지 확인
산출물
학습 계획, 관심 직무, 팀 협업 준비
Level 2

5주 SW 부트캠프

온라인 집중 과정에서 개발 기본기, Git 협업, API 연동, 배포 흐름을 프로젝트 중심으로 익힙니다.

목표
개발자로 일하는 기본 루프 습득
산출물
주차별 과제, 팀 프로젝트 데모
Level 3

테커 그룹 프로젝트

고성과자는 테커 그룹에 합류해 실제 서비스 수준의 프로젝트를 수행하고 포트폴리오를 고도화합니다.

목표
실전형 프로젝트 경험 축적
산출물
배포 서비스, GitHub 기록, 발표 자료
Level 4

커리어 완성 과정

코딩테스트, 이력서, 기술 면접을 준비하며 프로젝트 경험을 채용 담당자가 이해할 수 있는 언어로 정리합니다.

목표
지원 가능한 개발자 프로필 완성
산출물
이력서, 포트폴리오, 면접 답변 구조
Level 5

실리콘밸리 경험과 네트워킹

현지 기업 탐방, 엔지니어 네트워킹, 글로벌 개발 문화 경험을 통해 커리어 시야를 넓힙니다.

목표
글로벌 기준과 커리어 관점 확장
산출물
네트워크, 회고, 장기 성장 계획
Level 6

멘토 성장과 리더십

취업 이후에는 후배를 돕는 멘토, 시니어 멘토, 디렉터 역할로 성장하며 커뮤니티에 기여합니다.

목표
개발자 이후의 리더십 성장
산출물
멘토링, 기술 세션, 운영 기여
Process

부트캠프에서 전문 개발자까지

테커는 참가자가 다음 단계로 이동할 수 있도록 명확한 마일스톤을 둡니다.

1

SW 부트캠프 참가

5주 집중 온라인 부트캠프에서 개발 몰입 경험을 시작합니다.

2

테커 그룹 합류

고성과자는 테커 그룹 활동에 참여해 더 깊은 프로젝트와 커뮤니티를 경험합니다.

3

팀 프로젝트 수행

실제 서비스 수준의 팀 프로젝트로 개발자 포트폴리오를 완성합니다.

4

취업 준비 완성

코딩테스트, 이력서, 기술 면접을 실전 중심으로 준비합니다.

5

전문 개발자로 성장

취업 이후에도 멘토, 시니어 멘토, 디렉터로 성장하며 커뮤니티에 기여합니다.

Evaluation

다음 단계로 이동하는 기준

테커는 결과만 보지 않습니다. 문제를 정의하는 방식, 팀과 소통하는 태도, 피드백을 반영하는 속도까지 함께 봅니다.

Project Output기능 구현, 배포, 발표까지 이어지는 실제 결과물
Code Quality읽기 쉬운 코드, Git 사용, 리뷰 반영, 문제 해결 과정
Teamwork역할 책임, 커뮤니케이션, 일정 관리, 동료 피드백
Career Readiness이력서, 포트폴리오, 인터뷰에서 경험을 설명하는 능력

경력 있는 신입

프로젝트 경험과 커뮤니케이션 역량을 함께 갖춘 지원자로 성장합니다.

선순환 멘토링

먼저 성장한 멤버가 다음 기수의 멘토가 되어 커뮤니티의 기준을 높입니다.

지속적인 자기개발

취업 이후에도 기술 세션과 네트워킹으로 학습 흐름을 유지합니다.
